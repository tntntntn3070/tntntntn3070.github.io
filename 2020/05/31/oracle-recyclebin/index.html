Recycle Bin

테이블이 삭제되면 완전히 삭제되지 않고 테이블의 이름을 변경 후 테이블스페이스 내의 Recycle Bin 영역에 이동 (Oracle 10g 이상 지원)

FLASHBACK TABLE - Recycle Bin 으로 옮겨가는 테이블
FLASHBACK TABLE 이름 규칙 - BIN\$unique_id\$version

Session 레벨에서 Recycle Bin 활성화 여부 설정 (기본 : Enabled)
